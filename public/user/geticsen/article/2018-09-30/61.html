这是第一次写自己的博客

顺便写一些自己构建自己的博客的经历


构建自己博客之前一定要学习的足够扎实 构建博客涉及的只是比较多，不要急一步一步来 下面说一下博客的建立过程
  • 1.网页的学习  html  js css  还有一些框架 学完之后要求是可以自己把一个模板改造成自己想要的样子
  • 2.网站的建设语言 java python php 都可以  但是基础一定要扎实
  • 3.网络的学习  至少要了解到 http/ip协议  还有网络的层次结构
  • 4.linux的学习这是一个漫长的过程  对于服务器经验的为零的我来说  慢慢做只能说很无奈
  • 5.域名等的认识注册自己的域名，租服务器
  • 6.前面四点做完  学习一下数据库   你就可以制定自己的博客网站模板了  不太建议 你直接去用WordPress等工具 自己慢慢做会收获很多，刚开始的时候进度是非常慢的 这时候不要急慢慢的将之前学的一点点加到你要做的博客系统之中，在这个漫长的学习，以及设计过程中你将会发现自己各方面的能力都在稳步提升。
  • 最后将本地的 网站发布到自己的服务器
